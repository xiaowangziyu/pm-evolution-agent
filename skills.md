# PM Skills Marketplace - 产品经理技能库

完整的 phuryn/pm-skills 技能体系，包含 65 个技能和 36 个工作流，基于 Teresa Torres、Marty Cagan、Alberto Savoia 等大师的方法论。

---

## 八大模块概览

| 模块 | 技能数量 | 核心内容 |
|------|---------|---------|
| 产品发现 | 13 | 创意、假设、实验、OST、访谈 |
| 产品战略 | 12 | 愿景、商业模式、定价、竞争分析 |
| 产品执行 | 15 | PRD、OKR、路线图、冲刺、回顾 |
| 市场研究 | 7 | 画像、细分、旅程、规模、竞品 |
| 数据分析 | 7 | 北极星、漏斗、留存、实验、仪表盘 |
| 营销增长 | 8 | 增长漏斗、获客激活留存、推荐、内容 |
| 上市推广 | 6 | 上市计划、Beta、新闻稿 |
| 工具包 | 6 | 问题定义、原型、可用性测试 |

## 1. 产品发现 (pm-product-discovery)

### 技能 (13 个)

| 技能 | 说明 |
|------|------|
| brainstorm-ideas-existing | 现有产品多角度头脑风暴（PM/设计师/工程师视角） |
| brainstorm-ideas-new | 新产品初始发现阶段创意头脑风暴 |
| brainstorm-experiments-existing | 为现有产品设计验证假设的实验 |
| brainstorm-experiments-new | 为新产品设计精益预型实验（Alberto Savoia） |
| identify-assumptions-existing | 识别现有产品的价值/可用性/可行性/商业风险假设 |
| identify-assumptions-new | 识别新产品的8类风险假设（包含上市、战略、团队） |
| prioritize-assumptions | 基于 Impact × Risk 矩阵对假设排序，给出实验建议 |
| prioritize-features | 基于影响、工作量、风险、战略匹配度排序功能列表 |
| analyze-feature-requests | 分析和分类客户功能需求，按主题和战略匹配度整理 |
| opportunity-solution-tree | 构建机会解决方案树（Teresa Torres）- 结果→机会→解决方案→实验 |
| interview-script | 创建结构化的用户访谈脚本，含 JTBD 探究问题 |
| summarize-interview | 将访谈摘要整理为 JTBD、满意度信号和行动项 |
| metrics-dashboard | 设计产品指标仪表盘，含北极星指标、输入指标和告警阈值 |

### 命令 (5 个)

| 命令 | 说明 |
|------|------|
| /discover | 完整发现循环：创意→假设映射→优先级排序→实验设计 |
| /brainstorm | 多角度头脑风暴（创意/实验 × 现有/新） |
| /triage-requests | 分析和排序批量功能需求 |
| /interview | 准备访谈脚本或摘要（准备/摘要） |
| /setup-metrics | 设计产品指标仪表盘 |

### 命令 (5 个)

| 命令 | 说明 |
|------|------|
| /strategy | 创建完整的 9 部分产品战略画布 |
| /business-model | 探索商业模式（精益/完整/初创/价值主张/全部） |
| /value-proposition | 用 6 部分 JTBD 模板设计价值主张 |
| /market-scan | 宏观环境分析（SWOT + PESTLE + 波特五力 + 安索夫） |

## 工作流组合

### 从 0 到 1 产品发现流程

1. **创意生成** → `/brainstorm ideas new`
2. **假设识别** → `identify-assumptions-new`
3. **假设排序** → `prioritize-assumptions`
4. **实验设计** → `/brainstorm experiments new`
5. **用户访谈** → `/interview prep`
6. **机会树构建** → `opportunity-solution-tree`
7. **指标设置** → `/setup-metrics`

### 从创意到上线全流程

1. **问题定义** → `problem-framing`
2. **战略规划** → `/strategy`
3. **商业模式** → `/business-model startup`
4. **价值主张** → `/value-proposition`
5. **PRD 写作** → `/write-prd`
6. **OKR 规划** → `/plan-okrs`
7. **路线图** → `/transform-roadmap`
8. **冲刺规划** → `/sprint plan`
9. **事前风险** → `/pre-mortem`
10. **上市计划** → `go-to-market-plan`
